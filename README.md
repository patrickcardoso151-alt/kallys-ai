# KALLYS AI

KALLYS AI is an experimental web-based tactical AI interface designed to orchestrate large language models with visualization tools and operator analysis.

## Overview

KALLYS AI combines a conversational interface with operational dashboards, memory systems and data visualization to explore how AI assistants could be used in analytical or tactical environments.

## Features

- Conversational AI interface
- Long-term memory summarization
- Operator profile analysis
- OCR image processing
- File ingestion
- Data visualization (charts)
- Voice input support
- Telemetry monitoring

## Technologies

- HTML
- CSS
- JavaScript
- Chart.js
- Tesseract.js
- LLM API integration
